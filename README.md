# RyzenBOX

A powerful TV box featuring the ASRock 4x4 7840U-1U motherboard

THIS PROJECT WILL BE A SUBMISSION TO HACKCLUB MACONDO. I AM NOT AN EMPLOYEE AT A COMPANY AND I AM A STANDALONE DESIGNER

NOTE: the motherboard described is the ASRock Industrial 4x4, featuring the AMD Ryzen 7840U CPU and the listed specifications below. the board can be seen on ASRock's official website or at https://www.asrockind.com/en-gb/4X4-7840U-1U

Front I/O features:
1. 2 USB4 (Type-C) ports, DisplayPort 1.4a output
2. 1 USB3.2 Gen2 (Type-A)
3. 1 combo audio jack (microphone and headphone)

Back I/O features:
1. 2 HDMI 2.0 ports (supports 4096x2160 @ 60Hz)
 Note: Depending on whether you are looking at the Mini PC (BOX) or the bare Motherboard (SBC), some variants list HDMI 1.4b
2. 1 x 2.5 Gigabit LAN port (Realtek controller)
3. 1 x 1 Gigabit LAN port (Realtek controller, supports DASH on some board variants)
4. 2 USB2.0 Type-A ports
5. 1 DC Jack (supports flexible 12V~19V DC-in)
<img width="442" height="196" alt="Screenshot 2026-06-15 214722" src="https://github.com/user-attachments/assets/80511998-1f04-426d-a923-3cfcd82a9e62" />

Internals:
1. M.2 Key-M: 1 slot (2242/2260/2280) for PCIe Gen4 x4 NVMe SSDs
2. M.2 Key-E: 1 slot (2230) pre-allocated for a Wi-Fi/Bluetooth module
3. SATA: 1 x SATA 3.0 port for an additional 2.5-inch drive
4. RAM: 2 x 262-pin DDR5 SO-DIMM slots (up to 5600MHz, max 96GB depending on module sizes)
